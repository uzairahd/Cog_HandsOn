# JUnit Setup Example

## Objective
Set up JUnit in a Java project and create a simple unit test.

## Tools Used
- Java
- Maven
- JUnit 4.13.2
- Visual Studio Code

## Project Structure

```
src/
 ├── main/
 │   └── java/
 │       └── com/example/App.java
 └── test/
     └── java/
         └── com/example/
             ├── AppTest.java
             └── CalculatorTest.java
```

## Test Case

The CalculatorTest verifies the addition of two numbers.

Expected:

```
5 + 3 = 8
```

## Result

- JUnit configured successfully.
- All tests executed successfully.
- Tests Passed: 2/2
